# Aggregate Functions

1) MIN
2) MAX
3) AVG
4) SUM
5) COUNT



![Desi.](img/min.png)


# MAX
• Checks all the rows and shows maximum
one. 
```
SELECT MAX(salary)
FROM employees
```

# AVG
• Add all rows and get
average. 
```
SELECT AVG(salary) 
FROM employees
```


# SUM
• Add all rows and shows SUM. 
```
SELECT SUM(salary)
FROM employees;
```
